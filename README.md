# VR Evacuation Experiment — Source Data & Statistical Results

> 虚拟现实疏散实验 — 源数据与统计结果

## Important Notice for Reviewers / 审稿人须知

This repository has been prepared for the peer review of a manuscript submitted to **Advanced Engineering Informatics**. In the spirit of data transparency, we publish the raw experimental data, publication-quality figures with detailed data reports, and statistical analysis results here for reviewers and readers to inspect, verify, and reproduce.

本仓库为期刊 **《Advanced Engineering Informatics》** 投稿的同行评审工作而专门整理。我们秉持数据公开、透明的原则，将本研究所涉及的实验原始数据、手稿核心图表与数据报告、以及统计分析结果一并发布于此，供审稿人与读者查阅、校验与复现。

## Repository Overview / 仓库概览

| Directory / 目录 | Content / 内容 |
|-----------|---------|
| `RawData/` | Raw experimental data (eye-tracking & movement trajectories per trial) / 实验原始数据（每试次的眼动追踪与运动轨迹） |
| `Figures/` | Publication-quality figures (PNG) with detailed data reports (CSV/TXT) / 手稿核心图表高清版及详细数据报告 |
| `Statistics/` | Statistical results referenced in the manuscript (XLS/XLSX) / 手稿所涉及统计结果 |

### Privacy Notice / 隐私声明

All participant identity information has been anonymized via **SHA-256** hashing. Questionnaire data and personal experience records are not included in this public disclosure and are available upon separate written request to the corresponding author.

所有受试者身份信息均已通过 **SHA-256** 哈希算法模糊化处理。问卷及个人经历数据不在本次披露范围内，可另行向通讯作者申请。
