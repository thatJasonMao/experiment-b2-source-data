# VR 疏散实验源数据与统计结果 — 完整文档

## 一、关于本仓库

本仓库为期刊 **《Advanced Engineering Informatics》** 投稿的同行评审工作而专门整理。我们秉持数据公开、透明的原则，将本研究所涉及的实验原始数据、手稿核心图表及其数据报告、以及统计分析结果一并发布于此，以便审稿人与读者查阅、校验与复现。

本次研究围绕虚拟现实（VR）火灾疏散实验中，烟气浓度与非玩家角色（NPC）数量对人员疏散决策与视觉注意行为的影响展开。实验采集了多名受试者在不同烟雾浓度与 NPC 密度条件下的眼动追踪数据与运动轨迹数据。

## 二、仓库结构

本仓库由以下三个主要部分构成：

### 📁 RawData — 实验原始数据

`RawData/` 目录下包含所有受试者在各实验试次中的原始采集数据。每个试次以 `Trial_<SHA256-Hash>` 命名，内含两个 CSV 文件：

| 文件 | 内容说明 |
|------|----------|
| `EyeGaze_Simulation_Info_*.csv` | 该试次中受试者的眼动追踪数据，包含时间戳、注视点坐标、注视目标等字段 |
| `Subject_Simulation_Info_*.csv` | 该试次中受试者的运动轨迹数据，包含时间戳、位置坐标、速度等字段 |

### 📁 Figures — 手稿核心图表（高清版）及数据报告

`Figures/` 目录下按图表类别组织，提供手稿中所有核心图表的 PNG 高清版本，并附带详细的数据报告文件（CSV/TXT 格式），便于审稿人核验图表背后的原始数据。

| 子目录 | 内容说明 |
|--------|----------|
| `GazeSpatialPattern/` | 各决策点（DP1-DP4）注视空间分布模式：DBSCAN 聚类结果与 KDE 热力图，附带 `Reports/` 下的聚类统计报告 |
| `Hesitations/` | 不同烟雾浓度与 NPC 数量条件下的犹豫点空间分布图，附带 `Reports/` 下的犹豫点坐标数据 |
| `SpeedPattern/` | 各决策点的速度分布密度热力图、平面速度矢量图、轨迹速度曲线及整体速度分布分析 |
| `Trajectories/` | 不同烟雾浓度与 NPC 数量条件下受试者的疏散轨迹对比图 |

### 📁 Statistics — 手稿所涉及统计结果

`Statistics/` 目录下存放手稿中所有统计分析的最终结果数据。

| 子目录 | 文件 | 内容说明 |
|--------|------|----------|
| `Behaviors/` | `BehavioralFeatures.xls` | 受试者行为特征统计 |
| `Choices/` | `DirectionalChoices.xlsx` | 方向选择统计数据 |
| `FixationDuration/` | `FixationDuration_Mean.xlsx` / `FixationDuration_Total.xlsx` | 注视时长的均值与总计统计 |
| `GlobalMetrics/` | `EvacuationEfficiency.xlsx` | 疏散效率全局指标 |
| `PathMetrics/` | `PathEfficiency.xlsx` | 路径效率指标 |
| `Saccade/` | `SaccadeCount.xlsx` | 眼跳次数统计 |

## 三、数据隐私与伦理声明

为保护受试者隐私并遵循科研伦理规范，本仓库已执行以下脱敏措施：

- **受试者身份模糊化**：所有受试者的个人身份信息（包括姓名、年龄、性别等）已基于 **SHA-256** 哈希算法进行单向模糊化处理。各试次文件夹名 `Trial_<Hash>` 中的哈希值即由受试者身份信息与试次编号共同生成，无法逆向还原为原始身份。
- **数据披露范围限制**：涉及受试者个人经历、心理量表问卷（如 PANAS、SSQ、GSE、ITS 等）及其他可能间接识别个人身份的敏感数据，**不包含在本次数据披露范围内**。如需获取上述数据，请另行向通讯作者提出书面申请，我们将在符合科研伦理与数据保护法规的前提下酌情提供。
